DROPS第一次分享收获

这一个星期我主要在学习ctfhub上的技能树中的内容，虽然只得到了一些“定式的”结论，也愿意与大家分享一下。

软件

1 BurpSuite

在Web前置技能中，我简单地了解了如何使用BurpSuite进行抓包等操作。而在密码口令中，我更加了解了它的爆破功能。在个人浅显的了解中，爆破似乎是一种效率较低的方法，它通过穷举的方式，并且需要提前准备好密码字典，并且似乎只能破解一些较为简单的密码。不过我并不了解为什么它是通过通过Length来判断爆破是否成功。(Length不同的为答案)[爆破成功的用户名和密码返回长度与失败的返回长度差异很大。]？

2 Notepad++

在某道题中，我通过Writeup得知flag可直接通过Notepad++打开得出答案。

Motepad++是一种代码编辑器，功能较为丰富。我同时也在使用sublime。

信息泄露

我在信息泄露这一章节中了解了多种信息泄露的方式。即 Git SVN HG泄露。并发现解这类题都会有一个通法。

disearch的使用

通常，我们会首先进行disearch的扫描，根据个人的使用经验，颜色不同的 就是出了问题的。

而后，我们使用githack进行工具 clone 目标源代码到本地。(我找的githack似乎有问题，所以我使用的是Git_Extract)。随后，我们可以根据情况再度判断。一般情况下。我们可以首先尝试使用notepad++打开文件尝试是否可显示出flag。

值得强调的是，我们可以用git log 查看历史记录。从而判断之前做过什么操作。
